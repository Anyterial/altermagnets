The *Altermagnets Database* is an online interactive web application to navigate and search a dataset of materials computationally predicted to be altermangets, created by a team spanning [Lund University](https://www.lu.se/) (LU), [Luleå University of Technology](https://www.ltu.se/) (LTU), and [Linköping University](https://liu.se/) (LiU). The dataset was generated using the high-throughput toolkit, [*httk*](https://httk.org/). *For more info see our [about page](/about).*

Use the search panel to the left to search over combinations of chemistry, symmetry provenance, and spin-splitting thresholds.
